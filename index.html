This is a collection of frontend projects that I have made whilst following the freecodecamp online course. APIs and full-stack apps to follow!

Ransom Quotes

Threaten yourself inspired with this ransom note style random quote generator. The forismatic api is used to generate the quotes.

Naughts and Crosses

Play naughts and crosses against a simple minimax algorithm. The squares color co-ordinate on the computer turn to show the algorithm results.

Wedge Timer

This pomedoro-style timer allows you to split time into work chunks and breaks, each customisable between 1 minute and 1 hour.

Vanilla Calculator

A calculator implemented in plain javascript and CSS. Featuring automatic closing brackets and a scrollable history which can be edited and reused.

As you have probably already been able to tell, I'm a big fan of freecodecamp. You can use the map on the website to guide yourself through building your own portfolio. With the support available on this site, I have started using Node.js and deploying to Heroku, which I couldn't envision doing previously

Learn introductory computer science as it's taught at Havard. You can follow the course completely free, or upgrade to a verified version. You'll learn a little C, but the biggest selling point for me is the enthusiastic introduction to basic computer science topics.

On CodeWars, you can choose from a massive collection of Kata, that get harder as you improve your Javascript (or Python, Ruby, Clojure, Java...). Best of all; once you have submitted a working solution, you can see how other members have solved it, allowing you to pick up new tricks.
